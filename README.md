## [GDS系统知识点](/学习/GDS系统知识点.md)
GDS系统全称是气体检测系统，我们主要做的是可燃和有毒气体的检测，常见的检测气体有甲烷CH4，硫化氢H2S，氰化氢HCN,氨气NH3等，主要由前端探测器+后端控制器构成，传输方式有4-20ma，RS485两种方式，搭建上位机管理平台就可以监控了，这个文档是结合国家标准整理出来的相关知识点。
